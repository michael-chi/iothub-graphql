- Query Device with DeviceTwins
```
//  Query
query devices_query($input: [IoTHubDeviceInputType!]!) {
  devices(input: $input) {
    deviceId
    etag
    deviceTwins {
      deviceId
      moduleId
    }
    capabilities {
      iotEdge
    }
    modules {
      moduleId
      etag
      connectionState
    }
  }
}
//  Input
{
  "input": [
     {"deviceId": "device002"}
  ]
}
```
